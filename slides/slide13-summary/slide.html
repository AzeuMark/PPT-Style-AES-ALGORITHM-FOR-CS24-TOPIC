Summary

AES is a symmetric block cipher processing 128-bit blocks
2
Supports key sizes of 128, 192, or 256 bits
3
Each round applies SubBytes → ShiftRows → MixColumns → AddRoundKey
4
Multiple rounds of substitution, shifting, mixing, and key addition secure the data

AES is fast, proven secure, and used everywhere — HTTPS, Wi-Fi, file encryption
